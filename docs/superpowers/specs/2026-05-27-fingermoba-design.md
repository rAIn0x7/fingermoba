## 一、项目概述

FingerMOBA 是一款对标《指尖刀塔》的竖屏单手操作 MOBA 手游，面向 Android 平台。  
游戏采用经典 DOTA 推塔规则，3v3 对战，三路地图，竖屏设计便于单手游玩。

**开发阶段：**
- **阶段一（Demo）：** 单机可玩，3v3 含 AI 对手，3 个英雄，三路地图，跑通核心系统
- **阶段二（MVP）：** 联网对战，账号系统，扩充英雄池，完整排位/匹配系统

本文档覆盖**阶段一 Demo**的完整设计。

## 二、核心玩法规则

### 胜负判定
先摧毁敌方**基地水晶**的一方获胜。

### 地图结构
- 三路（上路 / 中路 / 下路），竖向布局
- 我方基地在下，敌方基地在上
- 每路各有 **2 座防御塔** + 路底 **1 座基地水晶**
- Demo 阶段：共用**单一基地水晶**（不分三路各一座），摧毁即胜，后续版本再拆分三路水晶

### 小兵系统
- 每 **30 秒**从双方基地同步刷出一波小兵（近战兵 3 + 远程兵 1）
- 小兵沿路自动推进，遇敌自动攻击
- 每路小兵独立刷新

### 防御塔规则
- 攻击范围内优先攻击敌方英雄，无英雄时攻击小兵
- 塔有护甲，英雄攻击塔伤害降低（鼓励推兵线后打塔）
- 塔被摧毁后不重建

## 三、经济系统

| 来源 | 金币 |
|------|------|
| 最后一击击杀我方小兵（拒补） | 0（敌方不获得经验和金币） |
| 最后一击击杀敌方小兵 | +40 |
| 击杀敌方英雄 | +200 |
| 助攻敌方英雄 | +100 |
| 被击杀后掉落 | 身上金币的 50%（向下取整） |
| 摧毁防御塔 | +150（全队共享） |

金币用于赛中购买**装备**（Demo 阶段 4 件占位装备）。

### Demo 装备列表（占位）
| 装备 | 费用 | 效果 |
|------|------|------|
| 短剑 | 400 | +15 攻击 |
| 铁甲 | 500 | +200 血量 |
| 法杖 | 600 | +40 法术强度 |
| 速靴 | 300 | +15% 移速 |

## 四、英雄系统

### 通用成长
- 比赛中通过击杀小兵/英雄获得经验
- 最高等级 **Lv.10**，升级提升基础属性
- 每升 1 级：HP +80，攻击 +4，技能伤害 +5%

### 复活机制
- 英雄被击杀后在己方基地出生点复活
- 复活时间：`5 + (等级 × 2)` 秒（Lv.1=7s，Lv.10=25s）
- 复活后满血满蓝出场

### Demo 英雄（3 个）

#### 英雄 A — 守卫·铁壁（坦克）
**定位：** 前排肉盾，近战，发起冲锋  
**基础属性：** HP 1200，攻击 55，移速 330，攻速 0.8/s  
**技能：**
- **冲锋（主动）** CD 8s：向目标方向冲刺 4 格，第一个命中的敌人被击晕 1 秒
- **铁壁（主动）** CD 12s：获得护盾（吸收 300 伤害），持续 4 秒

#### 英雄 B — 刺客·暗影（爆发输出）
**定位：** 爆发刺杀，近战，高风险高回报  
**基础属性：** HP 800，攻击 80，移速 360，攻速 1.1/s  
**技能：**
- **闪现刺杀（主动）** CD 10s：瞬移到目标背后，造成 200% 攻击力物理伤害
- **连斩（主动）** CD 6s：快速对目标连击 3 次，每击造成 60% 攻击力伤害

#### 英雄 C — 法师·风暴（范围控制）
**定位：** 后排法师，远程，团战核心  
**基础属性：** HP 700，攻击 45，移速 310，攻速 0.7/s  
**技能：**
- **闪电球（主动）** CD 7s：发射一枚闪电球，命中目标造成 280 法术伤害并传链给附近 2 个单位（各 140 伤害）
- **暴风眼（主动）** CD 15s：在目标区域释放风暴，持续 3 秒，范围内每秒造成 120 法术伤害

## 五、操控设计（竖屏单手）

```
┌─────────────────────┐
│  [小地图]    [设置]  │  ← 顶部 HUD
│                     │
│                     │
│     游戏画面         │  ← 中部战场（可滑动视角）
│                     │
│                     │
│  [摇杆区]  [普][A][B]│  ← 底部操控区
└─────────────────────┘
```

- **左下** 虚拟摇杆：控制英雄移动（浮动摇杆，手指放哪就出现在哪）
- **右下** 按钮区：`普攻` + `技能A` + `技能B`
- **点击敌人**：锁定目标（锁定后普攻自动追击）
- **技能方向**：部分技能点按钮后拖拽选择方向，松手释放（与指尖刀塔一致）

## 六、AI 系统（Demo 阶段）

AI 使用简单行为树，三个 Bot 各自独立决策：

```
根节点
├── 血量 < 20%？→ 撤退回基地
├── 有技能 CD 好了且有目标？→ 释放技能
├── 敌方英雄在攻击范围？→ 攻击英雄
├── 敌方小兵在攻击范围？→ 补刀
└── 默认 → 沿路推进
```

Bot 选英雄策略：3 个 Bot 自动各选一个不同定位的英雄（坦克/刺客/法师）。
